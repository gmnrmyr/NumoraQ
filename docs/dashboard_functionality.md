# Dashboard Functionality (Advanced Dashboard)

## Overview
The NUMORAQ dashboard is a comprehensive financial management platform that provides real-time tracking, analysis, and projections for all aspects of personal finance. The dashboard operates in two modes: **Simple Mode** for beginners and **Advanced Mode** for power users.

## Core Features

### 1. Portfolio Management

#### Liquid Assets Tracking
- **Real-time Asset Management**: Track cash, investments, crypto, and other liquid assets
- **Multiple Asset Types**: Support for manual entry, crypto (auto-price), stocks (live price), REITs/FIIs, precious metals, EVM wallets, and NFTs
- **Live Price Updates**: Automatic price fetching for supported assets
- **Asset Categories**: Financial, Investment, Tech, Physical, Business, and Lifestyle icons
- **Active/Inactive Toggle**: Enable or disable assets from calculations
- **Value Tracking**: Real-time portfolio value calculations

#### Illiquid Assets Management
- **Real Estate Tracking**: Property values, rental income, renovation status
- **Collectibles & Valuables**: Art, jewelry, vehicles, and other illiquid assets
- **Portfolio Summary**: Total asset breakdown with percentages
- **Asset Status**: Track rented, renovating, or planned properties

#### Blockchain Wallet Integration
- **Multi-Chain Support**: Bitcoin, EVM (Ethereum, Polygon, etc.), Solana
- **Real-time Token Tracking**: Automatic token balance and value updates
- **DeFi Position Monitoring**: Track yield farming, liquidity pools, and staking
- **NFT Portfolio**: Monitor NFT collections and floor prices
- **Wallet Management**: Add, remove, and refresh wallet data

### 2. Income Tracking

#### Active Income Management
- **Employment Income**: Track salary, bonuses, commissions
- **Freelance Income**: Multiple income sources with individual tracking
- **Business Income**: Revenue from entrepreneurial activities
- **Status Management**: Active/inactive income streams
- **Editable Values**: Real-time income amount updates
- **Progress Tracking**: Visual progress bars for income goals

#### Passive Income Tracking
- **Investment Dividends**: Stock dividends, REIT distributions
- **Rental Income**: Property rental revenue
- **Interest Income**: Savings account, bond interest
- **Royalty Income**: Intellectual property, licensing
- **Automated Calculations**: Monthly passive income totals

### 3. Expense Management

#### Recurring Expenses
- **Fixed Monthly Costs**: Rent, utilities, subscriptions
- **Categorized Tracking**: Housing, transportation, healthcare, etc.
- **Due Date Management**: Track payment schedules
- **Status Control**: Active/inactive expense management
- **Budget Allocation**: Visual expense breakdown

#### Variable Expenses
- **Flexible Spending**: One-time or irregular expenses
- **Date-Specific Tracking**: Schedule future expenses
- **Category Organization**: 40+ expense categories
- **Monthly Calculations**: Automatic variable expense totals

#### Expense Analytics
- **Category Breakdown**: Visual expense distribution
- **Monthly Totals**: Recurring + variable expense calculations
- **Budget vs Actual**: Track spending against budgets
- **Expense Trends**: Historical spending analysis

### 4. Debt Management

#### Debt Tracking
- **Multiple Debt Types**: Credit cards, loans, mortgages
- **Payment Status**: Pending, partial, paid tracking
- **Due Date Management**: Payment schedule monitoring
- **Interest Calculations**: Debt cost analysis
- **Progress Tracking**: Visual debt payoff progress

#### Debt Analytics
- **Total Debt Overview**: Sum of all outstanding debts
- **Payment Prioritization**: High-interest debt identification
- **Debt-to-Income Ratio**: Financial health metrics
- **Payoff Timeline**: Estimated debt-free date

### 5. Task Management

#### Financial Task Organization
- **Priority Levels**: Critical, urgent, high, medium, low
- **Categories**: Personal, financial, asset, goal-based tasks
- **Due Date Tracking**: Deadline management
- **Completion Status**: Task progress monitoring
- **Financial Impact**: Tasks with monetary consequences

#### Task Features
- **Drag & Drop**: Reorder tasks by priority
- **Filtering**: Category and priority-based filtering
- **Search**: Quick task finding
- **Bulk Operations**: Mass task management
- **Financial Impact Tracking**: Tasks affecting income/expenses
